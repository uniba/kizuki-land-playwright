# Playwright test for Kizuki Land 

## 1. Test flow
1. Upload a video
2. Create an event and select uploaded video from Step 1

## 2. Dependencies
1. Playwright
2. dotenv

## 3. Instructions
1. Create a '.env' file and set these environment variables:
    1. KIZUKI_EMAIL=xxx
    2. KIZUKI_PASSWORD=xxx

2. Run `npx playwright test`
